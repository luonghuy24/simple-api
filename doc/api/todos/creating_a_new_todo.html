Todos API

Creating a new todo

POST /todos

First, create an order, then make a later request to get it back

Parameters

Name Description
todo[title] Title of todo
todo[created_by] Created_by

Response Fields

Name Description
todo[id] Id of todo
todo[created_by] Id of creator
todo[title] Title of todo

Request

Headers

Host: example.org
Content-Type: application/x-www-form-urlencoded
Cookie:

Route

POST /todos

Body

{"todo":{"title":"new todo","created_by":1}}

cURL

curl "http://localhost:3000/todos" -d '{"todo":{"title":"new todo","created_by":1}}' -X POST \
	-H "Host: example.org" \
	-H "Content-Type: application/x-www-form-urlencoded" \
	-H "Cookie: "

Response

Headers

Content-Type: application/json; charset=utf-8
Cache-Control: no-cache
X-Request-Id: d6f3c67a-70e9-4e8a-ac0d-1b8bc012f5a2
X-Runtime: 0.004394
Content-Length: 80

Status

422 Unprocessable Entity

Body

{
  "message": "Validation failed: Title can't be blank, Created by can't be blank"
}
